<h1>
  PayrollSys-32 ARM Assembly Project
</h1>

<h2>
  Description
</h2>
<p> This is a minimal ARM Cortex-M4 microprocessor based modular environment, that can successfully perform following tasks:</p>
<ul>
  <li> Stores Individual Employee Informations into compact structures</li>
  <li> Uses one unique data.s file to avoid data leakage </li>
  <li> Computes overtimes, attendances, leave deficits, etc.</li>
  <li> Performs tax computations by slab system</li>
  <li> Generates alerts for anomalies</li>
  <li> Sorts employee informations based on net salary</li>
  <li> Generates full-payslip using ITM Ports</li>
</ul>

</h2>
